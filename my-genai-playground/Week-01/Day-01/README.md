# 🚀 Day 01 - Introduction to Generative AI

## 📌 Overview

Today’s focus was to understand the **fundamentals of Generative AI**, including AI models, their examples, applications, and key technical concepts like parameters, inputs, and performance requirements.

---

## 🧠 Topics Covered

### 🔹 What is an AI Model?

An AI model is a trained system that learns patterns from data and makes predictions or decisions based on that learning.

---

### 🔹 Examples of AI Models

* Chatbots (text generation)
* Image recognition systems
* Recommendation systems (Netflix, YouTube)
* Voice assistants

---

### 🔹 Applications of AI

* Healthcare (disease prediction)
* Finance (fraud detection)
* E-commerce (product recommendations)
* Education (personalized learning)
* Autonomous systems (self-driving cars)

---

## 🔑 Key Concepts

### 🔹 Model Parameters

* Parameters are internal values (weights) that the model learns during training
* Large models can have millions or even billions of parameters
* More parameters usually mean better learning capacity

---

### 🔹 Inputs

* The data given to the model
* Example:

  * Text → for chat models
  * Image → for vision models
* Quality and clarity of input directly affect output

---

### 🔹 Performance Requirements

#### 1. Accuracy

* How correct the model’s output is

#### 2. Latency

* How fast the model responds

#### 3. Scalability

* Ability to handle large number of users or data

#### 4. Efficiency

* Resource usage (CPU, GPU, memory)

---

## 🎯 Key Learnings

* AI models learn from data and make predictions
* Different models are used for different applications
* Parameters define model intelligence
* Inputs and performance factors play a critical role
* Understanding these basics is essential before moving to advanced topics
