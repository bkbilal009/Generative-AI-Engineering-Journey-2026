
# 🚀 Day 04 - ChatGPT Process, NLP Basics & Pre-trained Models

## 📌 Overview

Today’s focus was on understanding how ChatGPT processes information, along with key Natural Language Processing (NLP) concepts like stemming, lemmatization, and pre-trained models.

---

## 🧠 Topics Covered

### 🔹 ChatGPT Model Process

* Input text is given by the user
* Text is converted into tokens (tokenization)
* Model processes tokens using learned patterns
* Output is generated step-by-step

---

### 🔹 Stemming

* Reduces words to their root form
* Example:

  * playing → play
  * running → run
* Fast but less accurate

---

### 🔹 Lemmatization

* Converts words into meaningful base form
* Example:

  * better → good
  * ate → eat
* More accurate and context-aware

---

### 🔹 Pre-trained Models

* Models trained on large datasets before use
* Save time and computational resources
* Can be fine-tuned for specific tasks

---

## ⚙️ Key Learnings

* ChatGPT works using tokenization and prediction
* NLP techniques help in processing text efficiently
* Stemming vs Lemmatization difference is clear
* Pre-trained models are powerful and widely used
